# Flatmate From Hell

A chaotic flat-sim comedy game where you survive living with increasingly absurd roommates.

## Development

### Setup
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## Game Structure

- **Scenes**: Different game modes and screens
- **Flatmates**: Individual character behaviors and AI
- **Systems**: Core game mechanics (chaos, power-ups, saves)
- **UI**: Reusable interface components
- **Data**: JSON configuration files for game content

## Tech Stack

- Phaser 3 (Game Engine)
- TypeScript
- Webpack (Build System)
